三.while循环

while 循环会在指定条件为真时循环执行代码块。

在 do-while 中，do 内的语句块先执行一次，再判断条件。

这种形式的语法很少使用，除非你希望不管条件是否为真，循环体 至少执行一次。通常我们更倾向于使用另一个形式：while(…) {…}。

四.break和continue语句

  • break： 语句用于跳出循环。
  • continue： 用于跳过循环中的一个迭代。

五.label

label 是一个标签，可以使用 break 或 continue 使程序跳转到这个标签处执行（执行：break 或 continue），从而改变程序的执行流程。

循环过程为：i=0，j=0，i+j!=6.i=0,j=1,i+j!=6……i=0,j=6,满足条件

js数据类型

基本类型

  • 字符串（String）：用于字符串
  • 布尔(Boolean)：用于 true 和 false。
  • 数字(Number)：用于任何类型的数字——整数或浮点数，
  • 空（Null）：用于未知的值 —— 只有一个 null 值的独立类型。
  • 未定义（Undefined）：用于未定义的值 —— 只有一个 undefined 值的独立类型。
  • Symbol：用于唯一的标识符。

注：Symbol 是 ES6 引入了一种新的原始数据类型，表示独一无二的值。

引用数据类型

  • 对象(Object)
  • 数组(Array)
  • 函数(Function)
  • 两个特殊的对象：正则（RegExp）和日期（Date）。

一.String类型

1.1字符串的表示：

JavaScript 中的字符串必须被括在引号(单引号、双引号、反引号)里。

  • 双引号和单引号都是“简单”引用，在 JavaScript 中两者几乎没有什么差别。

  • 反引号是 功能扩展的引号。它们允许我们通过将变量和表达式包装在 ${…} 中，来将它们嵌入到字符串中。

    ${…} 内的表达式会被计算，计算结果会成为字符串的一部分。可以在 ${…} 内放置任何东西：诸如名为 name 的变量，或者诸如 1 + 2 的算数表达式，或者其他一些更复杂的。

1.2字符串的length属性：

  • 通过对 String 类型 “打点” 的方式，可以调用其内置属性。
  • 字符串的 length 属性表示字符串的长度。

1.3字符串的常用方法:

“方法” 就是能够打点调用的函数，字符串类型有丰富的内置方法。

  • charAt():得到指定位置字符
  • substring():提取子串
  • substr():提取子串
  • slice():提取子串
  • toUpperCase():将字符串变为大写
  • toLowerCase():将字符串变为小写
  • indexOf():检索字符串
  • trim():删除首尾空格
  • trimStart():删除首部空格
  • trimEnd():删除尾部空格

二.Boolean类型

boolean 类型仅包含两个值：true 和 false。

使用 typeof 运算符可以检测值或者变量的类型。

布尔类型在 关系运算 和 逻辑运算 中广泛运用。

三.Number类型

在 JS 中，所有数字不分大小、不分整浮、不分正负，都是数字类型。

四.Null类型

  • null 表示 “空”，可以理解为它是 “空对象”。

当我们需要将对象销毁、数组销毁或者删除事件监听时，通常将它们设置为 null。

用 typeof 检测 null 结果为 object。

js 函数

一.函数的定义

1.1函数声明方式（function关键字）

  • ()：圆括号中是形参列表,以逗号隔开，上述为空。
  • {}：花括号内为函数语句块，即函数体。

1.2函数表达式（匿名函数、函数字面量）

  • function(){}：匿名函数

六.函数的参数和返回值

6.1 函数的参数

  • 调用函数时传入 “实际参数”

注意：JS 只有 “值传递” 没有 “引用传递”，对于复杂类型的传递，传递的不是引用，而是那个变量里面的值（即引用的地址）。

引用传递：修改形参，实参也会改变。JS 中复杂类型的实参是个地址值不需要改变，也改变不了，改变的是地址所指向的堆中的复杂类型的具体值，此处具有迷惑性，要加以辨别。

6.1.1 arguments

  • 函数内 arguments 表示它接收到的实参列表，它是一个类数组对象。

  • 类数组对象：所有属性均为从 0 开始的自然数序列，并且有 length 属性，和数组类似可以用方括号书写下标访问对象的某个属性值，但是不能调用数组的方法。

6.2 函数的返回值

函数体内可以使用 return 关键字表示 “函数的返回值”。

调用一个有返回值的函数，可以被当做一个普通值，从而可以出现在任何可以书写值的地方。

遇见 return 即退出函数。

结合 if 语句的时候，往往不需要写 else 分支了。

js表达式与操作符

发表于 2022-07-20 分类于 JavaScript学习笔记

隐式类型转换

如果参与数学运算的某操作数不是数字类型，那么 JS 会自动将此操作数转换为数字型。

隐式转换的本质是 JS 内部自动调用 Number() 函数

IEEE754

在 JS 中，有些小数的数学运算不是很精准。

JS 使用了 IEEE754 二进制浮点数算术标准，这会使一些个别的小数运算产生 “丢失精度” 问题。

注：不是所有的小数运算都会出现精度问题，只是少数部分。

  • 解决方法：
    在进行小数运算时，要调用数字的 toFixed() 方法保留指定的小数位数。

    幂和开根号

    JS 中没有提供幂运算、开根号的运算符，需要使用 Math 对象的相关方法进行计算。

  • Math.pow(a, b)：求 a 的 b 次方。

  • Math.sqrt(a)：求 a 的平方根。

向上取整和向下取整

  • Math.ceil()：向上取整。

  • Math.floor()：向下取整。

  • Math.round()：把一个数字舍入为最接近的整数（“四舍六入”，“五不一定”）

注意：向上、向下的标准是：X轴正方向为上！

负 ———— 0 ————> 正

相等和全等

  • 两个等号 == 运算符，不考虑值的类型，它会进行隐式转换后比较值的字面量是否相等。

  • 三个等号 === 运算符，不仅比较值是否相同，也比较类型是否相同。

null 和 undefined 用 == 进行比较涉及隐式强制类型转换，ES5 规范中规定。

=== 比较为 false，是因为 null 与 undefined 类型不同。

建议没有特殊情况请一律使用 ===

NaN不自等

NaN 作为一个特殊的数字类型值，它在用 == 比较的时候也有特殊的结果。
