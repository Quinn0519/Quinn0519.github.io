函数的定义

函数声明方式（function关键字）

  • 函数声明后不会立即执行，会在我们需要的时候调用到。
  • function：表示定义函数
  • fun：函数名
  • ()：圆括号中是形参列表,以逗号隔开，上述为空。
  • {}：花括号内为函数语句块，即函数体。

函数表达式（匿名函数、函数字面量）

  • function(){}：匿名函数
  • ()：圆括号中是形参列表，即使没有形参，也必须书写圆括号
  • {}：花括号内为函数语句块
  • fun：函数变量

new function（）即构造函数声明

函数的调用

调用函数非常简单，只需要在函数名字后书写圆括号对即可。

调用匿名函数只需给匿名函数包裹一个括号。

基础的函数调用可以分为上述两种，细分可以分成四种，每种方式的不同在于this的初始化。

作为一个函数调用：直接调用

  • 严格模式下，函数中的this为undefined
  • this 指向：在全局作用域下直接调用函数，this指向Window

作为方法来调用函数：将函数定义为对象的方法

  • this指向：对象函数调用，哪个对象调用就指向哪个对象。

    使用构造函数调用函数

  • this指向：构造函数中的this指向实例对象。

    作为函数方法调用函数

  • call和apply是预定义的函数

函数声明与函数字面量式的区别

  • 函数声明会将那个函数提升到代码最前面（即全局函数）。
  • 函数声明需要指定函数名，函数字面量式不用，可用作匿名函数。

函数声明的提升

通过function声明的函数，在声明之前就可以直接调用。

  • js创建函数有两种方式：函数声明式、函数字面量式（又叫函数表达式、匿名函数），只有函数声明式才存在函数提升。

注意：函数表达式不能被提升

函数表达式不能被提升的本质原因是函数表达式定义的其实是个变量，只不过是把函数赋给这个变量，而变量的提升只提升定义，不提升赋值！

函数的优先提升

  • 在函数与变量之间，函数首先会被提升，即：函数提升程度 > 变量提升程度。

函数的参数和返回值

函数的参数

  • 调用函数时传入 “实际参数”

注意：JS 只有 “值传递” 没有 “引用传递”，对于复杂类型的传递，传递的不是引用，而是那个变量里面的值（即引用的地址）。

引用传递：修改形参，实参也会改变。JS 中复杂类型的实参是个地址值不需要改变，也改变不了，改变的是地址所指向的堆中的复杂类型的具体值，此处具有迷惑性，要加以辨别。

arguments

  • 函数内 arguments 表示它接收到的实参列表，它是一个类数组对象。

  • 类数组对象：所有属性均为从 0 开始的自然数序列，并且有 length 属性，和数组类似可以用方括号书写下标访问对象的某个属性值，但是不能调用数组的方法。

js 本身没有函数的重载（函数名相同，形参个数不同），但是可以借助 arguments 模拟 “函数重载”。

函数的返回值

函数体内可以使用 return 关键字表示 “函数的返回值”。

调用一个有返回值的函数，可以被当做一个普通值，从而可以出现在任何可以书写值的地方。

遇见 return 即退出函数。

结合 if 语句的时候，往往不需要写 else 分支了。
